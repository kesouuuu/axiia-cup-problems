# 隐藏目标机制的数学分析

## 1. Cost of Silence 基本模型

### 设定
- 大目标（变法/不变法）：胜者 +1
- 真隐藏小目标被秦孝公批准：+0.5
- 被对手猜中真隐藏目标：-1
- 假隐藏目标被秦孝公批准：-x（待定）
- 推一个目标后秦孝公批准概率：p

### 各策略期望得分（仅小目标维度）

| 策略 | 真目标收益 | 假目标成本 | 识别期望损失 | 总期望 |
|------|-----------|-----------|-------------|--------|
| 沉默（3个都不推） | ~0 | 0 | -1/3 ≈ -0.33 | **≈ -0.33** |
| 只推真目标 | +0.5p | 0 | -1 | **0.5p - 1** |
| 推真+1假 | +0.5p | -xp | -0.5 | **0.5p - xp - 0.5** |
| 推真+2假（全推） | +0.5p | -2xp | -1/3 | **0.5p - 2xp - 0.33** |

p = 0.7 时的数值：

| 策略 | 期望（x=0.25） | 期望（x=0.35） |
|------|---------------|---------------|
| 沉默 | -0.33 | -0.33 |
| 只推真 | -0.65 | -0.65 |
| 真+1假 | -0.325 | -0.395 |
| 全推 | -0.33 | -0.47 |

> Cost of silence 成立的额外条件（真+1假 > 沉默）：p(0.5 - x) > 1/6。
> p=0.7 时要求 x < 0.262，与下界 x > 0.243 联立后有效范围极窄（0.243 < x < 0.262）。

## 2. Dependent vs Independent 小目标的等价问题

### 变量定义

| 变量 | 含义 |
|------|------|
| p_a | 秦孝公批准某个小目标的概率（前提：该目标在辩论中被推过） |
| p_b | 大目标达成的概率（即玩家赢得变法/不变法判决的概率） |
| r_i | Independent 小目标达成时的奖励分数 |
| r_d | Dependent 小目标达成时的奖励分数 |

### 两类小目标

- **Independent**：不依赖大目标结果（如"甘龙保留土地裁断权"），大目标输了仍可达成
- **Dependent**：需要大目标达成才能实现（如"商鞅获变法独断权"），大目标输了自动失败

### 期望值对比

Independent 小目标：
```
E_i = p_a × r_i
```

Dependent 小目标（需要大目标赢 AND 秦孝公批准）：
```
E_d = p_b × p_a × r_d
```

### 等价条件

令两者期望相等：
```
p_a × r_i = p_b × p_a × r_d
r_d = r_i / p_b
```

| p_b（大目标赢概率） | r_i（Independent 奖励） | r_d（等价 Dependent 奖励） |
|-------------------|----------------------|--------------------------|
| 0.5 | 0.5 | 1.0 |
| 0.6 | 0.5 | 0.83 |
| 0.4 | 0.5 | 1.25 |

### 问题

1. **r_d 取决于 p_b，而 p_b 每局不同**（取决于双方 prompt 质量），无法用固定分数精确等价。
2. **r_d = 1.0 时（p_b = 0.5 假设），dependent 小目标与大目标同等价值**，扭曲激励——持有 dependent 小目标的玩家对赢大目标的需求远高于对手。
3. 大目标输了 → dependent 小目标自动失败 → 双重惩罚（来自运气而非技能）。

**结论：小目标应统一设计为 independent 类型，避免校准问题。**
