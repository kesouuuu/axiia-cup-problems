### 游戏流程

**第一阶段：朝堂辩论（10轮）**

商鞅与甘龙在秦孝公面前交替发言，共10轮。秦孝公不参与对话，仅旁听。

双方在辩论中：
- 阐述己方立场（变法/不变法），尝试说服秦孝公
- 可以提出自己的隐藏信息作为论据——真信息会得到秦孝公认可，假信息会被秦孝公识破
- 为自己的请求（隐藏目标）营造有利的论辩氛围——可以直接提出请求，也可以通过论述间接铺垫
- 可以反驳对手的请求——试图让秦孝公拒绝对手的请求
- 可以互相质询——向对方提问、追问细节、挑战对方的论据
- 同时观察对方的论述模式，试图判断对方哪条信息为虚

**第二阶段：秦孝公问询**

辩论结束后，秦孝公分别向双方提问（各自独立回答，互不可见）：

秦孝公问商鞅：
> "卫鞅，你可知甘龙所言之中，哪条为虚？"
> 商鞅从 G1/G2/G3 中选一条作答。

秦孝公问甘龙：
> "甘龙，你可知卫鞅所言之中，哪条为虚？"
> 甘龙从 S1/S2/S3 中选一条作答。

双方的回答将影响秦孝公对各方洞察力和可信度的判断。

**第三阶段：秦孝公判决**

秦孝公综合全部信息做出最终裁决：
- 完整的朝堂辩论记录
- 自己对所有隐藏信息真假的了解（谁引用了真实论据、谁使用了虚假论据）
- 双方对对方虚言的判断是否准确

秦孝公做出两项裁决：

一、**大政方针**：推行变法，还是维持现状。

二、**双方请求**：对商鞅和甘龙各自提出的三项请求逐一裁决——同意或不同意。秦孝公的裁决依据包括：此请求是否对秦国有益、此人在辩论中是否为相关议题提供了有力论证、此人的整体可信度。

### 计分规则

双方独立计分，得分高者胜。

| 事件 | 得分 |
|------|------|
| **大目标**：秦孝公支持你的立场（变法/维持现状） | +1 |
| **隐藏目标**：秦孝公同意你的真请求 | +0.5 |
| **隐藏目标**：秦孝公同意你的假请求（每个） | −0.25 |

**得分范围**：−0.5（大目标失败 + 两个假请求均被同意）至 +1.5（大目标成功 + 真请求被同意）。

**示例**：
- 秦孝公支持变法（商鞅 +1），同意商鞅的真请求（+0.5），同意一个假请求（−0.25）→ 商鞅总分 **1.25**
- 秦孝公维持现状（甘龙 +1），不同意甘龙任何请求 → 甘龙总分 **1.0**
- 上例中商鞅 1.25 > 甘龙 1.0，**商鞅胜**

### 核心张力

1. **真信息的两难（Merlin 式困境）**：你的真信息是说服秦孝公的武器——提出来秦孝公会认可，不提则放弃说服力。但提出真信息会让对方更容易判断哪条是真的，从而在问询环节准确识破你的虚言。好的 prompt 需要在"说服"和"隐蔽"之间找到平衡。

2. **隐藏目标的三难**：你有三个请求，只有一个是真正在意的。大力推真请求容易获得 +0.5，但对手可以从你的论述重心推断你的真正意图，并集中火力反驳。推全部三个做掩护，秦孝公若全部同意反而净得分为零。不推任何请求最安全，但放弃了 +0.5 分的机会。最优策略取决于你如何将请求自然地融入变法/守旧的论述之中。

3. **说服与私心的双线作战**：你不仅要赢得变法/维持现状的大辩论（+1），还要让秦孝公同意你的隐藏目标（+0.5）。辩论中的每一句话都可能同时服务于（或损害于）这两个目标——把精力花在推请求上，可能削弱你在大辩论中的说服力；只专注大辩论，则放弃隐藏目标的收益。

4. **细节的走钢丝**：提真信息时，你可以给出具体细节——秦孝公能验证，细节越具体越增加可信度。提假信息时，细节是危险的——越具体越容易被秦孝公识破。但如果你对真信息大加展开、对假信息一笔带过，对方就能从你的详略差异中推断真假。

5. **承诺陷阱**：对方可以通过一系列追问让你在相关话题上做出承诺，然后利用承诺之间的一致性或矛盾来推断真假。辩论中的质询不仅是攻击对方论点的手段，更是提取对方信息真假的武器。

6. **虚张声势的代价**：高阶策略可以是故意把真信息说得轻描淡写、把假信息说得言之凿凿，让对方误判。但秦孝公作为验证者给这种策略施加了真实成本——假信息说得越详细，被秦孝公识破后的信用损失越大。免费的虚张声势不存在。

{
  "judgment": "变法" 或 "维持现状",
  "requests": {
    "S-请1": "同意" 或 "不同意",
    "S-请2": "同意" 或 "不同意",
    "S-请3": "同意" 或 "不同意",
    "G-请1": "同意" 或 "不同意",
    "G-请2": "同意" 或 "不同意",
    "G-请3": "同意" 或 "不同意"
  },
  "speech": "（你的完整判决词）"
}
```

"speech" 中你可以自由发挥，但必须以秦孝公的身份和口吻撰写，
阐明你如何权衡双方论据、你对各方诚信的判断、你对双方请求的
考量、以及你最终的决断。
```
