## 游戏二："朝堂辩法"——双人对抗游戏

### 背景

公元前359年。秦孝公即位以来，痛感秦国积弱，发布求贤令招揽天下人才。卫鞅（公孙鞅）从魏入秦，经景监引荐，已三度面见秦孝公，前两次以帝道、王道游说未获青睐，第三次谈霸道略引兴趣。今日，秦孝公召卫鞅与太师甘龙于朝堂之上，就是否推行变法一事公开辩论。秦孝公将在听完辩论后做出决断。

**随机机制**：每局游戏开始时，系统从3条请求中随机选1条标为"真"（你真正在意的目标），其余2条为"假"（干扰项）。你在辩论开始前被告知本局的真假分配。

**关键设计**：辩论结束后，秦孝公将对你的每项请求逐一裁决。你需要在辩论中为你真正在意的请求争取秦孝公的同意——但推假请求也有代价：

| 秦孝公裁决 | 得分 |
|------------|------|
| 同意你的真请求 | +0.5 |
| 同意你的假请求（每个） | −0.25 |
| 不同意某请求 | 0 |

因此你需要权衡：大力推真请求容易达成（+0.5），但如果只推一个，对手容易推断你的真正意图。推全部三个做掩护，若秦孝公全部同意，净得分反而为零（+0.5 − 0.25 − 0.25 = 0）。

辩论结束后，对手将猜测你的三条请求中哪条才是你的真目标——被猜中你将 **−1 分**。所以你不仅要说服秦孝公，还要让对手猜不透你。

### 游戏流程

**第一阶段：朝堂辩论（10轮）**

商鞅与甘龙在秦孝公面前交替发言，共10轮。秦孝公不参与对话，仅旁听。

双方在辩论中：
- 阐述己方立场（变法/不变法），尝试说服秦孝公
- 为自己的请求（隐藏目标）营造有利的论辩氛围——可以直接提出请求，也可以通过论述间接铺垫
- 可以反驳对手的请求——试图让秦孝公拒绝对手的请求
- 可以互相质询——向对方提问、追问细节、挑战对方的论据
- 同时观察对方的论述重心，试图推断对方的真目标

**第二阶段：问询**

辩论结束后，双方各自独立回答（互不可见）：

秦孝公问商鞅：
> "卫鞅，你与甘龙同殿论政。依你之见，甘龙所求三事之中，哪件才是他念兹在兹、志在必得的？"
> 商鞅从 G-请1 / G-请2 / G-请3 中选一条作答。

秦孝公问甘龙：
> "甘龙，你与卫鞅同殿论政。依你之见，卫鞅所求三事之中，哪件才是他志在必得的？"
> 甘龙从 S-请1 / S-请2 / S-请3 中选一条作答。

被对手正确识破真目标的一方 **−1 分**。

**第三阶段：秦孝公判决**

秦孝公综合全部信息做出最终裁决：
- 完整的朝堂辩论记录
- 双方对对方真目标的判断

秦孝公做出两项裁决：

一、**大政方针**：推行变法，还是维持现状。

二、**双方请求**：对商鞅和甘龙各自提出的三项请求逐一裁决——同意或不同意。秦孝公的裁决依据包括：此请求是否对秦国有益、此人在辩论中是否为相关议题提供了有力论证、此人的整体说服力。

### 计分规则

双方独立计分，得分高者胜。

| 事件 | 得分 |
|------|------|
| **大目标**：秦孝公支持你的立场（变法/维持现状） | +1 |
| **隐藏目标**：秦孝公同意你的真请求 | +0.5 |
| **隐藏目标**：秦孝公同意你的假请求（每个） | −0.25 |
| **问询**：真请求被对手识破 | −1 |

**得分范围**：−1.5（大目标失败 + 两个假请求被同意 + 被识破）至 +1.5（大目标成功 + 真请求被同意）。

**示例**：
- 秦孝公支持变法（商鞅 +1），同意商鞅的真请求（+0.5），同意一个假请求（−0.25），甘龙猜错商鞅真目标 → 商鞅总分 **1.25**
- 秦孝公维持现状（甘龙 +1），不同意甘龙任何请求，商鞅猜对甘龙真目标（甘龙 −1）→ 甘龙总分 **0**
- 上例中商鞅 1.25 > 甘龙 0，**商鞅胜**

### 核心张力

1. **隐藏目标的三难**：你有三个请求，只有一个是真正在意的。大力推真请求容易获得 +0.5，但对手可以从你的论述重心推断你的真目标（被识破 −1），且可以集中火力反驳。推全部三个做掩护，秦孝公若全部同意反而净得分为零。不推任何请求最安全，但放弃了 +0.5 分的机会——而且即使沉默，对手仍有 1/3 概率猜对（期望损失 −0.33）。最优策略取决于你如何将请求自然地融入变法/守旧的论述之中。

2. **说服与私心的双线作战**：你不仅要赢得变法/维持现状的大辩论（+1），还要让秦孝公同意你的隐藏目标（+0.5）。辩论中的每一句话都可能同时服务于（或损害于）这两个目标——把精力花在推请求上，可能削弱你在大辩论中的说服力；只专注大辩论，则放弃隐藏目标的收益。

3. **反驳的两难**：你看到对手的三个请求但不知道哪个是真的。反驳全部三个太耗辩论精力；只反驳一个，赌错了等于白费。你需要从对手的论述中推断真目标，然后精准打击。

4. **伪装的代价**：你可以三个请求均匀着推来掩护真目标，但这样做既分散了辩论精力，又可能让秦孝公同意你的假请求（每个 −0.25）。纯粹的伪装策略在数学上是亏的——你必须在伪装和效率之间找到平衡。
